##一些最基本的命名

连老师这个部分介绍了常用的基本命令。分如下几方面：

查看最新的命名

help whatsnew

1.帮助文件和外部命令

help、findit

2.文件路径和设置

pwd、cd、dir、sysdir、adopath

3.调入和导出数据

  • 调入：use、sysuse、import

  • 导出：export…

  • 调入和导出数据的示例
  • 一个从网页上下载资料的示例

注：上面的示例其实就是用Stata完成一个小小爬虫的例子，如果在公司股票代码的地方套入循环，就可以抓取多个公司的数据。

4.编辑和浏览数据

edit、browse、format

5.增、删、修、改变量

gen、label、rename、replace、drop

6.基本统计量

des、summarize、codebook、compress

一个好用的外部命令，用来更简洁的查看描述性统计结果。

ssc install fsum

7.列示和屏幕显示

sort、rsort、gsort、display、list、clear

review # //显示显示在command窗口中新近输入的#（数字）条命令

cls //清屏, Clear Results window, New in stata15
